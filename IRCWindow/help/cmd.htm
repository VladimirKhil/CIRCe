Команды

Команды серверу вводятся в строке ввода. Все команды начинаются с символа '/'. Цирцея сама реагирует на несколько команд.

 /msg <Получатель> <Текст> аналогична /PRIVMSG <Получатель> :<Текст> — упрощённый синтаксис для отправки сообщений;

 /echo <Текст> — выводит текст в окно чата без его отправки в сеть;

 /splay <Файл> — проигрывает звуковой или видеофайл. Файл задаётся либо по абсолютному пути, либо просто по имени. В последнем случае он должен находиться внутри папки Media Цирцеи (неважно, непосредственно в ней или внутри глубоко вложенной в неё директории).
