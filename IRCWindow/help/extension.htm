Расширенные возможности

В пункте главного меню «Настройки» содержатся возможности для дополнительного управления программой. Горячие клавиши позволяют настроить стандартные сообщения, которые будут выдаваться на канал после нажатия соответствующей клавиши. Палитра цветов позволяет определить собственное отображение цветов в чате. Заметьте, что Цирцея поддерживает 99 цветов в отличие от остальных клиентов IRC, разнообразие которых ограничено 16 цветами. Пункт меню «Оформление» позволяет настроить под себя внешний вид программы.

 Пункт «Дополнения» позволяет запускать программы-аддоны, написанные специально для Цирцеи. Такую программу может написать любой желающий. Для этого требуется лишь создать аддон на платформе .NET и реализовать внутри библиотеки аддона необходимый интерфейс. После этого любой желающий сможет поместить этот аддон в папку AddOns Цирцеи и запускать его при необходимости. Аддоны поддерживают и автоматический запуск при входе в программу. Такие аддоны можно использовать для более тщательной настройки пользовательского интерфейса под конкретные задачи.

При желании все аддоны можно исключить и убрать их из меню.

Конкретное руководство по написанию аддонов будет размещено отдельно.

Цирцея поставляется с одним встроенным аддоном, предназначенным для проведения SIGame в Интернете. Этот аддон заменяет технически устаревшие скрипты для программы mIRC, которые также позволяли проводить СИ в IRC. Новый аддон по сравнению со скриптами обладает следующими преимуществами:

1) регулярные автоматические обновления Цирцеи обновляют и аддон, исправляя ошибки в нём;

 2) все необбходимые для проведения игры команды вынесены прямо в окно чата, что избавляет ведущего от необходимости переключать окна во время проведения СИ. В штатном режиме ведущий использует лишь одно окно - окно чата. При необходимости каких-то поправок, вызванных ошибочными действиями ведущего, он открывает вспомогательное окно и вносит исправления. Во вспомогательном окне находятся и настройки аддона;

3) аддон расширяет контекстное меню пользователей на канале. Теперь с помощью этого меню можно добавить пользователй в таблицу прямо из окна канала;

4) во время игры после обращения к нажавшему кнопку игроку аддон выводит ведущему на канал правильный ответ;

5) ход времени, отсчитываемого таймером, визуально отображается в виде заполняющейся полоски в окне чата;

6) аддон непосредственно работает с файлами вопросов в формате SIQ и XML, что избавляет ведущего от необходимости конвертировать их в дополнительный формат (как это было в случае со скриптами).
